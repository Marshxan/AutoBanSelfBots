# AutoBanSelfbots
A bot that automatically detects selfbots and ban them

## How to install it?


*  Install dependencies  
`npm install`

*  Edit the config file  
Modify the `config.js` file with your informations! (token, log channel)

*  Run the bot  
`node index.js`

And it's done!
